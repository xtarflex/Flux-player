# Security Policy

## Supported Versions

Currently, Flux Player is in early development. We provide security updates for the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| < 0.2.0 | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please send an email or contact the core maintainers directly. We will try to address the issue as soon as possible and provide an estimated timeline for a fix.

Please provide:
* A description of the vulnerability.
* Steps to reproduce.
* Potential impact.
